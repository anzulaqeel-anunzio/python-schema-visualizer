# Database Schema Visualizer (Python)

A tool to generate [PlantUML](https://plantuml.com/) diagrams from a simple schema definition (JSON/YAML).

## Features

-   **Schema to PlantUML**: Converts a schema definition into Element-Relationship Digram (ERD) syntax.
-   **Visualize Relationships**: Automatically draws lines between foreign keys.
-   **Lightweight**: Generates text output that can be rendered anywhere PlantUML is supported.

## Installation

1.  **Clone the repository**
2.  **Install Dependencies**
    ```bash
    pip install -r requirements.txt
    ```

## Usage

1.  Define your schema in `schema.json`.
2.  Run the visualizer:
    ```bash
    python run_visualizer.py
    ```
3.  Copy the output to [PlantText](https://www.planttext.com/) or any PlantUML viewer.
